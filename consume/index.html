Bienvenido!

Esta aplicacion tiene 2 funcionalidades. Validar el RUT con formato Chileno y dar formato a un nombre ingresado con sus 2 apellidos y genero.

Las funciones de esta aplicacion fueron desarrolladas en Java, con su framework Spring Boot. Esto nos permite generar una API Rest para poder desplegarla en Tomcat y despues consumirla con Fetch API de JavaScript. En cuanto al front-end, fue desarrollado solo con HTML-CSS-JS y el framework Bootstrap 4

Si quieres utilizar alguna de estas 2 funciones no dudes en hacer clic en alguno de los botones de abajo!
